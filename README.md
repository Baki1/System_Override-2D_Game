# If It Works, Don't Touch It 🪄👾

> *"A perfectly working system is only one touch away from a catastrophic glitch."*

## About The Project
This 2D narrative-driven game was developed as a term project for the **Visual Programming** course at **Bandırma Onyedi Eylül University**. It explores a unique meta-concept where the player's interaction intentionally breaks a flawless digital system. 

The core philosophy of the game revolves around the famous developer motto: **"If it works, don't touch it."**

### 📖 The Concept & Lore
In a perfectly balanced digital ecosystem, an external interference (the player's touch) causes a critical system failure—a "glitch." From this anomaly, our protagonist, the **Glitch Wizard**, is violently pulled out of their natural state. Forced into a corrupted environment, the Wizard must navigate through the broken logic of the system, trying to escape the chaos or find a way to patch the very fabric of their reality. 

## Academic Context
* **Institution:** Bandırma Onyedi Eylül University
* **Course:** Visual Programming
* **Objective:** Applying core programming principles and object-oriented design to create an interactive 2D experience.

## 🎮 How to Play
You don't need to compile the source code to try the game! You can directly download the playable version:
1. Go to the [Releases](../../releases) section on the right side of this repository.
2. Download the latest `.zip` file.
3. Extract the folder and run `CalisiyorsaDokunma_2D_Game.exe`.
